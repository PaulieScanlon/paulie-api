## Twitter

This endpoint uses the Twitter `users/by/username/${username}` endpoint under the hood to return information about a Twitter user.

[Run in browser](/get-twitter-user)

<Box
  sx={{

>

| Endpoint                | Type | Parameters       | Allowed Domains          | Response |
| ----------------------- | ---- | ---------------- | ------------------------ | -------- |
| `/api/get-twitter-user` | POST | username: string | https://paulie.dev       | JSON     |
| -                       | -    | -                | https://pauliescanlon.io | -        |

</Box>

<small>Example Request</small>

```javascript
// using axios
await axios('https://paulieapi.gatsbyjs.io/api/get-twitter-user', {
  method: 'POST',
  data: {
    username: 'PaulieScanlon',
  },
})

// using fetch
await fetch('https://paulieapi.gatsbyjs.io/api/get-twitter-user', {
  method: 'POST',
  body: JSON.stringify({ username: 'PaulieScanlon' }),
})
```
