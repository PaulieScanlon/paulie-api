## Twitter

### Latest Tweets

This endpoint uses the Twitter `users/${id}/tweets` endpoint under the hood to return a users latest Tweets. Use
[this link](https://tools.codeofaninja.com/find-twitter-id) to find your Twitter `id`.

| Endpoint                 | Type | Parameters | Required | Default | Response |
| ------------------------ | ---- | ---------- | -------- | ------- | -------- |
| `/api/get-latest-tweets` | POST | id: number | true     | n/a     | JSON     |

<RunInBrowser>
  <GetLatestTweets />
</RunInBrowser>

<RequestResponseExpand type='request'>

```javascript
// axios
await axios('https://paulieapi.gatsbyjs.io/api/get-latest-tweets', {
  method: 'POST',
  data: {
    id: 470012453

});

// fetch
await fetch('https://paulieapi.gatsbyjs.io/api/get-latest-tweets', {

});
```

## Stripe

### Make Stripe Payment

This endpoint uses the Stripe `stripe.checkout.sessions.create` method under the hood to create a checkout url for a
user provided payment amount.

| Endpoint                   | Type | Parameters          | Required | Default | Response |
| -------------------------- | ---- | ------------------- | -------- | ------- | -------- |
| `/api/make-stripe-payment` | POST | product: string     | true     | n/a     | JSON     |
| -                          | -    | amount: number      | true     | n/a     | -        |
| -                          | -    | success_url: string | true     | n/a     | -        |
| -                          | -    | cancel_url: string  | true     | n/a     | -        |

<RunInBrowser>
  <MakeStripePayment />
</RunInBrowser>

<RequestResponseExpand type='request'>

```javascript
// axios
await axios('https://paulieapi.gatsbyjs.io/api/make-stripe-payment', {
  method: 'POST',
  data: {
    product: 'prod_KAgqqzBEBmuYkT',
    amount: 5,
    success_url: 'https://www.mdx-embed.com',

    amount: 5,
    success_url: 'https://www.mdx-embed.com',
```
